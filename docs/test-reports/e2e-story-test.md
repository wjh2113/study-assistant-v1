## 🔍 发现的问题

### 高优先级

#### 1. AI 服务未配置
- **文件**: `backend/.env`
- **问题**: 缺少 `AI_API_KEY` 和 `AI_API_URL` 配置
- **影响**: AI 答疑功能不可用
- **建议**: 配置有效的 AI API 密钥
- **修复难度**: 低

#### 2. 练习页面空白
- **文件**: `frontend/src/pages/Practice.jsx` (缺失)
- **问题**: `/practice` 路由存在但页面无内容
- **影响**: 用户无法进行练习
- **建议**: 创建练习页面组件
- **修复难度**: 中

#### 3. 用户昵称未显示
- **文件**: `frontend/src/pages/Dashboard.jsx`
- **问题**: Dashboard 显示"你好，"后面为空
- **可能原因**: 用户数据中 `username` 字段为空或 API 返回格式不匹配
- **建议**: 检查注册流程和用户数据结构
- **修复难度**: 低

### 中优先级

#### 4. 数据统计不一致
- **问题**: Dashboard 显示"知识点总数：0"，但知识点管理页有 4 个知识点
- **可能原因**: `progressAPI.getStats()` 未正确计算
- **建议**: 检查统计 API 实现
- **修复难度**: 中

#### 5. 课本上传功能未完整测试
- **原因**: 浏览器自动化工具无法直接操作文件选择器
- **建议**: 手动测试或改进测试工具配置
- **修复难度**: 中
