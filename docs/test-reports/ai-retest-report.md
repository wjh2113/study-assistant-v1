## 🔧 修复的问题

在测试过程中发现并修复了以下问题：

### 1. AI Controller API 调用格式错误
**问题**: 原代码使用 OpenAI 格式的 API 调用，但配置的是阿里云 DashScope API

**修复**: 修改 `backend/src/controllers/aiController.js`，使用阿里云 DashScope 正确的请求格式：
```javascript
// 修复前
{
  model: 'gpt-3.5-turbo',
  messages: [...]
}

// 修复后
{
  model: 'qwen-plus',
  input: {
    messages: [...]
  }
}
```

### 2. 数据库路径配置错误
**问题**: `.env` 中 `DATABASE_PATH=file:./database/sqlite.db` 格式不正确

**修复**: 移除 `file:` 前缀
```
DATABASE_PATH=./database/sqlite.db
```

### 3. AI 问答记录表外键约束问题
**问题**: `ai_qa_records` 表的外键约束导致插入失败

**修复**: 重建表结构，移除不必要的外键约束

### 4. AIQARecordModel ID 生成方式错误
**问题**: 模型使用 UUID 字符串作为 ID，但数据库表使用 INTEGER AUTOINCREMENT

**修复**: 修改 `backend/src/models/AIQARecord.js`，使用数据库自增 ID
```javascript
// 修复前
const id = `aiqa_${Date.now()}_${Math.random().toString(36).substr(2, 9)}`;
const result = stmt.run(id, userId, question, answer, knowledgePointId);

// 修复后
const result = stmt.run(userId, question, answer, knowledgePointId);
return this.getById(result.lastInsertRowid);
```
