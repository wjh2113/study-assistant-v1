## [1.0.0] - 2026-03-15

### ✨ 新增功能

#### P0 - MVP 核心功能
- **用户系统**
  - 手机号验证码登录/注册
  - JWT Token 认证
  - 用户资料管理（学生/家长）
  - 角色权限控制（STUDENT/PARENT/ADMIN）

- **知识点管理**
  - 知识点 CRUD 操作
  - 知识点搜索功能
  - 知识点状态管理（ACTIVE/ARCHIVED/DRAFT）
  - 知识点分类和标签

- **学习进度跟踪**
  - 学习时长记录
  - 完成度跟踪
  - 学习统计分析
  - 学习连续性追踪

- **AI 答疑**
  - 文字提问
  - AI 回答生成
  - 问答历史记录
  - 问答记录搜索

#### P1 - 功能增强
- **AI 出题 (ISSUE-P0-003)**
  - 基于课本内容自动生成题目
  - 支持多种题型（选择题等）
  - 可配置题目数量和难度
  - AI 任务日志追踪

- **课本解析 (ISSUE-P1-002)**
  - PDF 课本上传
  - AI 目录识别
  - 异步解析处理（Worker）
  - 解析状态追踪

- **薄弱点分析 (ISSUE-P1-003)**
  - 知识点掌握度计算
  - 艾宾浩斯遗忘曲线集成
  - 薄弱点智能识别
  - 个性化学习建议

- **积分系统 (ISSUE-P1-004)**
  - 练习积分奖励
    - 答对题目：+10 分/题
    - 正确率≥80%：+20 分
    - 正确率=100%：+50 分（≥3 题）
    - 连续练习≥3 天：+5 分/3 天
  - 打卡积分奖励
    - 每日打卡：+5 分
    - 连续 7 天：+20 分
    - 连续 30 天：+100 分
  - 积分明细记录
  - 积分统计查询

- **排行榜 (ISSUE-P1-005)**
  - 总排行榜
  - 周排行榜
  - 月排行榜
  - 我的排名查询
  - 定时计算任务（node-cron）
  - Redis 缓存优化

- **文件上传 (ISSUE-P1-006)**
  - 本地文件存储
  - 支持图片/文档上传
  - 文件大小限制
  - 上传目录管理

- **速率限制 (ISSUE-P1-007)**
  - 全局速率限制：100 次/IP/15 分钟
  - 发送验证码：3 次/手机号/1 小时
  - AI 出题：20 次/用户/1 小时
  - 课本解析：10 次/用户/1 小时

- **日志系统 (ISSUE-P1-008)**
  - Winston 日志框架
  - 请求日志记录
  - 错误日志追踪
  - 审计日志（可选）

  - 清理 console.log（替换为条件日志）
  - 清理临时数据库文件

## 版本说明

### 语义化版本

- **主版本号 (Major)**: 不兼容的 API 变更
- **次版本号 (Minor)**: 向后兼容的功能新增
- **修订号 (Patch)**: 向后兼容的问题修正

### 版本类型

- **开发版**: 0.x.x - 功能开发中
- **正式版**: 1.x.x - MVP 完成，可生产使用
- **LTS 版**: 长期支持版本（未来规划）
