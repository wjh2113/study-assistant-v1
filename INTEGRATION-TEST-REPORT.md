## 🔍 根本原因分析

### 1. 认证系统问题（最严重）
**现象**: 所有需要认证的接口返回 401  
**原因**: 
- `authHelper.createAndLogin()` 依赖的 `/api/auth/send-code` 和 `/api/auth/register` 返回 400
- 测试模式下验证码验证逻辑可能有问题
- 数据库表未创建导致用户无法保存

### 2. 数据库初始化问题
**现象**: 数据库索引创建失败，提示表不存在  
**原因**:
- 测试前未执行数据库迁移
- `initDatabase()` 可能没有创建所有表

### 3. Redis Mock 问题
**现象**: 队列模块尝试连接真实 Redis  
**解决**: 已在 jest.setup.js 中 mock ioredis 和 bullmq
